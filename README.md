# With AU Daily

## Project Introduction

**With AU Daily** is a personal project aimed at building to introduce my girlfriend notion template. The project focuses on a clean, minimal user experience while leveraging modern frontend technologies to ensure good performance and scalability.

## Technologies Used
* React
* TypeScript
* SCSS / Tailwind CSS *(depending on project configuration)*
* Shadcn
* framer-motion

## Local Setup and Run

### 1. Clone the repository

```bash
git clone https://github.com/dokkazy/with-au-daily.git
cd with-au-daily
```

### 2. Install dependencies

```bash
npm install
# or
pnpm install
```

### 3. Run the project locally

```bash
npm run dev
# or
pnpm dev
```

### 4. Access the application

Open your browser and navigate to:

```
http://localhost:5173
```

## Notes

* Node.js LTS version is recommended.
* If environment variables are required, make sure to configure the `.env` file before running the project.
